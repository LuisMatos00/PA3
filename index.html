Descripción del Trabajo Práctico

El objetivo de este trabajo práctico es que se demuestre la capacidad para usar JavaScript en un sitio 
web con un proyecto de comercio  electrónico-carrito  de compras para lograr  la interactividad con el 
usuario y trabajar de manera proactiva en equipo.  Deberán desarrollar una página  web interactiva que 
haga uso de HTML, CSS y JavaScript, implementando diversas funcionalidades interactivas.

Instrucciones del Trabajo Práctico
1.	Desarrollo de la Página Web: Crear una página web interactiva que incluya los siguientes elementos:
  o	JavaScript Básico:
    - 	Uso de valores, tipos y operadores.
    - 	Implementación de estructuras de control.
  o	Funciones en JavaScript:
    - 	Definición y uso de funciones, funciones de flecha, argumentos, funciones recursivas y funciones crecientes.
  o	Estructuras de Datos:
    - 	Uso de objetos y arrays.
    - 	Implementación de encapsulamiento, métodos, prototipos, clases, mapas y polimorfismo.
  o	Manejo de Eventos:
    - 	Implementación de eventos y manipulación del DOM.
    - 	Manejo de eventos de teclado, desplazamiento, foco, carga y temporizadores.
2.	Estructura y Diseño: La página web debe tener una estructura coherente y un diseño atractivo y funcional.
3.	Entrega: Se deben entregar los siguientes archivos:
  o	Archivo HTML (.html).
  o	Archivo CSS (.css).
  o	Archivo JavaScript (.js).
